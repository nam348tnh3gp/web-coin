How WebCoin Works

Simple, transparent, and educational blockchain mechanics

1

Create Wallet

Generate a secure wallet with secp256k1 key pair. Your private key is encrypted with AES-256.

2

Get Mining Job

Connect to WebCoin server and receive a mining job with current block height and difficulty.

3

Mine the Block

Find a nonce that produces a SHA-1 hash with required leading zeros. Higher difficulty = more zeros.

SHA-1 Mining Algorithm 
bool checkDifficulty(uint8_t* hash, uint32_t diff) {
    for (uint32_t i = 0; i < diff; i++) {
        if (hash[i] != 0) return false;
    }
    return true;
}

// Block structure (52 bytes):
// [height:4B][prev_hash:20B][timestamp:4B][zero_hash:20B][nonce:4B]
// Hash must start with 'diff' number of zero bytes

Frequently Asked Questions

Everything you need to know about WebCoin

What is WebCoin?
WebCoin is a lightweight blockchain system designed for learning and experimentation. It features multi-platform mining (Arduino, PC, Web, Terminal), secp256k1 cryptography, and a complete web interface for wallet management.
How do I start mining?
Choose your platform: • Web: Visit https://webcoin-1n9d.onrender.com, create wallet, go to Mine tab • Arduino: Upload miner.ino, run python AVR_Miner.py • PC: Run python PC_Miner.py and follow interactive setup • Terminal: node src/miner.js <wallet> <private_key>
What mining algorithm does WebCoin use?
WebCoin uses SHA-1 for Proof-of-Work. Blocks are hashed with SHA-1, and the hash must start with 'difficulty' number of zero bytes. Transactions are signed with secp256k1 (same as Bitcoin).
What are the expected hash rates?
Hash rates by platform: • Arduino Uno/Nano: 50-80 kH/s • Arduino Mega: 80-120 kH/s • Arduino Due: 150-200 kH/s • ESP32: 200-500 kH/s • PC (Python): 500-2000 kH/s • Web Browser: 100-500 kH/s
How are mining rewards calculated?
Rewards are dynamic: reward = min_reward + (difficulty * (max_reward - min_reward) / 10). With min_reward=10 and max_reward=200, higher difficulty = higher rewards. This encourages mining when the network is more secure.
Is WebCoin secure?
Yes! WebCoin implements industry-standard security: secp256k1 for digital signatures, AES-256-CBC for private key encryption, bcrypt for password hashing, JWT for authentication, and Helmet/CORS/rate limiting for API protection.
